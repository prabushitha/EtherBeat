# EtherSci


## Dependencies
- SQLite3
- Rocksdb

## Quick Guide

### Build Sqlite and Rocksdb databases

You can easily build and parse ethereum blockchain by running EtherBuilder.

1. Goto `../tool/EtherBuilder`
2. Setup sqlite3 and rocksdb as mentioned in the Builder
2. Compile and run EtherBuilder

### Running EtherSci

`./run.sh`
